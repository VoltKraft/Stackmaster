# ui/

The Stackmaster single-page application. It consumes the core's REST
and WebSocket API and contains the graphical node editor.

**Status:** not yet implemented. The framework decision is deferred to
[ADR-0003](../docs/adr/0003-frontend-framework.md) and is coupled to
the node-editor library decision in
[ADR-0007](../docs/adr/0007-node-editor-library.md).

See [../docs/NODE_EDITOR.md](../docs/NODE_EDITOR.md) and
[../docs/ARCHITECTURE.md](../docs/ARCHITECTURE.md).
