# examples/

Concrete, fully-commented examples that illustrate Stackmaster's
configuration shape. Examples are **not** part of the shipped product;
they are documentation with a file extension.

- [`manifests/`](manifests/) — infrastructure manifests for
  `stackmaster apply -f`.
- [`workflows/`](workflows/) — workflow definitions that go with the
  manifests.

**Status:** placeholders only. Schemas are draft and may change
before v0.1.

See [../docs/WORKFLOWS.md](../docs/WORKFLOWS.md),
[../docs/PROVIDERS.md](../docs/PROVIDERS.md), and
[../docs/ARCHITECTURE.md](../docs/ARCHITECTURE.md).
